1. sh файлы для полуавтоматической установки archlike дистрибутивов;
2. Конфиг и список расширений для FireFox;
3. Список расширений и настройки для VSCodium;
4. Список расширений и настройки для Zed;
5. Конфиг для DoomEmacs;
6. Конфиг для NeoVim.

В дальнейшем будут добавляться/изменяться/удаляться какие-либо файлы/разделы в зависимости от использования.
