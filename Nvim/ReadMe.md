# Конфиг для Neovim.

Очередная попытка запилить долбанный комбаин, который может схавать все. И, скорее всего, как всегда не удачная.

## Установка
Само собой, нужно накатить сам nvim.
Потом скопировать файлы по нужному пути:

Linux: `.config\nvim`
Windows: `AppData\Local\nvim`
MacOS: `хз`

Запустить сам nvim и дождаться пока lazy закончит работать.
Радоваться что, возможно, все работает .. а может и нет .. как повезет, короче.
