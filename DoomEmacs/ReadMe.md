# Простой и, возможно, тупой конфиг для DoomEmacs.

## Установка
После того как была выполнена установка самого DoomEmacs, файлы нужно скопировать по пути

Windows: `;TLDR`
Linux: `.config/doom/`
MacOS: `Не знаю и знать не хочу`

Выполнить команду `sh doom sync` из папки `.config/emacs/bin`.
Дождаться окончания выполнения команды.
Запустить emacs.
